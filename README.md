# Solvexa — Company Website

Modern, animated company website for **Solvexa** — a digital solutions company offering websites and mobile applications.

## Features

- Responsive design (mobile, tablet, desktop)
- Smooth scroll animations (Framer Motion)
- Glass morphism UI with gradient accents
- Sections: Hero, Services, About, Portfolio, Process, Contact, Footer
- Interactive contact form
- Mobile-friendly navigation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Icons

## Customization

- Update contact info in `src/components/Contact.jsx`
- Change company stats in `src/components/About.jsx` and `src/components/Hero.jsx`
- Add real projects in `src/components/Portfolio.jsx`
- Update social links in `src/components/Footer.jsx`
